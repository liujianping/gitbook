# 快速开始

## 工具安装

````bash
$: npm i gitbook-cli -g
````

## 快速开始

````bash
# 样例项目
$: mkdir sample; cd sample
# 初始化目录
$: gitbook init
# 安装依赖
$: gitbook install
# 本地服务
$: gitbook serve
# 打包
$: gitbook build
````

## 项目下载

本项目地址: [github.com/liujianping/gitbook](github.com/liujianping/gitbook).可直接下载测试。

````
$: git clone https://github.com/liujianping/gitbook.git
$: cd guide
$: gitbook install
$: gitbook serve
````
